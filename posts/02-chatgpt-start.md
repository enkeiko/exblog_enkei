## ChatGPT가 뭐야?

**ChatGPT**는 OpenAI라는 회사가 만든 **대화형 AI**야. 쉽게 말해서 **뭐든지 대답해주는 AI 친구**라고 생각하면 돼.

### 왜 ChatGPT부터 배워?

| 이유 | 설명 |
|------|------|
| **가장 유명해** | 전 세계에서 가장 많이 쓰는 AI. 자료도 많고, 물어볼 사람도 많아 |
| **무료로 충분해** | 유료(월 $20) 안 해도 웬만한 건 다 돼 |
| **한국어 잘해** | 한국어로 질문하면 한국어로 대답해줘. 어색함 없음 |
| **범용적이야** | 글쓰기, 번역, 코딩, 공부, 아이디어... 뭐든 물어볼 수 있어 |

다른 AI(Claude, Gemini 등)도 좋지만, **입문용으로는 ChatGPT가 딱**이야.

## 1단계: ChatGPT 가입하기

### PC로 가입하기

**1. 사이트 접속**

브라우저 열고 주소창에 입력해:

```
https://chat.openai.com
```

또는 네이버/구글에서 **"ChatGPT"** 검색해서 들어가도 돼.

---

**2. 가입하기 클릭**

첫 화면에서 **"Sign up"** (가입하기) 버튼을 눌러.

이미 계정 있으면 **"Log in"** (로그인) 누르면 돼.

---

**3. 가입 방법 선택**

세 가지 방법 중 골라:

| 방법 | 추천도 | 설명 |
|------|--------|------|
| **Continue with Google** | ⭐⭐⭐ | 구글 계정으로 한 번에 가입. 가장 편함 |
| **Continue with Apple** | ⭐⭐ | 아이폰/맥북 쓰면 이것도 편해 |
| **이메일 가입** | ⭐ | 이메일 입력 → 비밀번호 설정 → 인증메일 확인 |

**구글 계정 추천**해. 클릭 한 번이면 끝이거든.

---

**4. 약관 동의 & 간단한 설정**

- 이용약관 동의 (체크하고 넘어가)
- 이름 입력 (본명 아니어도 돼)
- 생년월일 입력

끝! **계정 생성 완료**야.

---

### 스마트폰으로 가입하기

앱으로 쓰고 싶으면:

1. **앱스토어** (아이폰) 또는 **플레이스토어** (안드로이드) 열기
2. **"ChatGPT"** 검색
3. **OpenAI** 만든 앱 다운로드 (가짜 앱 조심! 개발사 확인해)
4. 앱 열고 위에서 설명한 대로 가입

**주의**: "ChatGPT" 이름 달고 있는 가짜 앱 많아. 꼭 **OpenAI**가 만든 건지 확인해.

## 2단계: 화면 구성 이해하기

가입 끝났으면 이제 화면을 살펴보자.

### 메인 화면 구성

```
┌─────────────────────────────────────────┐
│  ☰ 사이드바        ChatGPT              │
├─────────────────────────────────────────┤
│                                         │
│    [새 채팅]                            │
│    [이전 대화 목록]                      │
│    [이전 대화 목록]                      │
│                                         │
│                                         │
├─────────────────────────────────────────┤
│                                         │
│     💬 "무엇을 도와드릴까요?"            │
│                                         │
│  ┌─────────────────────────────────┐    │
│  │ 메시지를 입력하세요...         │ ➤  │
│  └─────────────────────────────────┘    │
│                                         │
└─────────────────────────────────────────┘
```

### 핵심 기능 설명

| 영역 | 설명 |
|------|------|
| **사이드바 (왼쪽)** | 이전 대화 목록. 새 대화 시작도 여기서 |
| **메인 영역 (가운데)** | 대화 내용이 표시되는 곳 |
| **입력창 (아래)** | 질문 입력하는 곳. 엔터 치면 전송 |
| **모델 선택 (위)** | GPT-4o, GPT-4o mini 등 선택 가능 |

### 무료 vs 유료 차이

| 구분 | 무료 | Plus ($20/월) |
|------|------|---------------|
| GPT-4o | ⭕ (제한적) | ⭕ (더 많이) |
| GPT-4o mini | ⭕ 무제한 | ⭕ 무제한 |
| 이미지 생성 | ❌ | ⭕ |
| 파일 업로드 | ⭕ (제한적) | ⭕ |
| 느린 시간대 속도 | 느림 | 빠름 |

**결론: 무료로 시작해.** 부족하다 느끼면 그때 유료 고민해도 늦지 않아.

## 3단계: 첫 대화 나눠보기

자, 이제 진짜 써보자!

### 첫 번째 대화

입력창에 이렇게 써봐:

```
안녕! 나는 AI를 처음 써보는 사람이야. 너는 뭘 할 수 있어?
```

엔터 치면 ChatGPT가 자기소개를 해줄 거야. 뭐라고 하는지 읽어봐.

### 대화의 핵심: "이어지는 맥락"

ChatGPT의 가장 큰 장점은 **대화가 이어진다**는 거야.

예를 들어:

```
[너] 파이썬이 뭐야?
[ChatGPT] 파이썬은 프로그래밍 언어로...

[너] 그거 배우기 어려워?
[ChatGPT] 파이썬은 초보자에게 추천하는 언어로... (아까 물어본 거 기억함!)
```

"그거", "아까 그거" 같은 말을 알아들어. 마치 사람이랑 대화하는 것처럼 자연스럽게 물어보면 돼.

## 바로 써먹는 프롬프트 5개

뭘 물어봐야 할지 모르겠지? 이거 복사해서 써봐:

### 1. 개념 쉽게 설명받기

```
"블록체인"이 뭔지 초등학생도 이해할 수 있게 설명해줘.
```

### 2. 글 초안 작성

```
회사 야유회 공지 이메일 초안 좀 써줘.
날짜: 12월 20일 금요일
장소: 한강공원
준비물: 편한 복장
```

### 3. 번역 + 뉘앙스 설명

```
"I'm not feeling well" 이거 한국어로 뭐야?
언제 쓰는 표현이고, 비슷한 표현도 알려줘.
```

### 4. 아이디어 브레인스토밍

```
20대 직장인 대상 부업 아이디어 10개 추천해줘.
초기 비용 적고, 퇴근 후에 할 수 있는 것으로.
```

### 5. 일정 정리

```
다음 내용을 깔끔한 표로 정리해줘:

- 월요일: 팀 회의 10시, 점심 약속 12시
- 화요일: 보고서 마감
- 수요일: 재택근무
- 목요일: 출장 (부산)
- 금요일: 프로젝트 발표 3시
```

## 실전 활용 팁

> 💡 **팁 1: 역할을 지정해줘**
>
> "넌 10년 경력 마케터야. 이 문구 봐줘" 라고 하면 더 전문적인 답변을 받을 수 있어.

> 💡 **팁 2: 구체적으로 물어봐**
>
> ❌ "여행 계획 짜줘" (너무 막연함)
> ✅ "12월 말 2박 3일 부산 여행 계획 짜줘. 20대 커플이고, 예산은 50만원이야"

> 💡 **팁 3: 마음에 안 들면 다시 시켜**
>
> "다른 버전으로 다시 써줘", "좀 더 짧게", "좀 더 친근하게" 이렇게 수정 요청하면 돼.

## 주의사항

AI 쓸 때 꼭 기억해:

- **개인정보 입력 금지** - 주민번호, 비밀번호, 계좌번호 절대 입력하지 마
- **100% 믿지 마** - AI도 틀릴 수 있어. 중요한 건 팩트체크 필수
- **회사 기밀 조심** - 업무 내용 입력할 때 민감한 정보는 빼고 물어봐
- **저작권 주의** - AI가 쓴 글을 그대로 쓰면 문제 될 수 있어. 수정해서 써

이건 나중에 5회차에서 더 자세히 다룰 거야.

## 자주 묻는 질문

### Q. 무료인데 뭔가 결제하라고 떠요

가입은 무료가 맞아. **"Upgrade to Plus"** 버튼은 유료 플랜 광고니까 무시해도 돼. X 버튼 누르고 그냥 써.

### Q. 영어로 대답해요

한국어로 질문했는데 영어로 대답하면, 그냥 이렇게 써:

```
한국어로 대답해줘
```

그 다음부터는 한국어로 대답할 거야.

### Q. 대화 내용이 저장되나요?

응, 왼쪽 사이드바에 저장돼. 나중에 이어서 대화할 수도 있고, 삭제할 수도 있어.

### Q. 하루에 몇 번까지 쓸 수 있어요?

무료 계정은 **GPT-4o**는 횟수 제한 있고, **GPT-4o mini**는 거의 무제한이야. 일상적인 질문은 mini로도 충분해.
