## 프롬프트가 뭐야?

**프롬프트(Prompt)**는 AI에게 보내는 질문이나 명령이야. 쉽게 말해서 "AI한테 하는 말" 전부가 프롬프트야.

근데 왜 이게 중요하냐면... 같은 AI라도 어떻게 물어보느냐에 따라 답변 퀄리티가 하늘과 땅 차이거든.

---

## 나쁜 프롬프트 vs 좋은 프롬프트

직접 비교해볼게.

■ 예시 1: 이메일 작성

**나쁜 프롬프트:**
```
이메일 써줘
```
→ AI: "무슨 이메일을 쓸까요?" (뭘 원하는지 모름)

**좋은 프롬프트:**
```
거래처에 보낼 미팅 일정 조율 이메일을 써줘.
- 받는 사람: ABC 회사 김과장님
- 내용: 다음 주 화요일 또는 목요일 오후 2시에 만남 제안
- 톤: 정중하지만 친근하게
```
→ AI: 바로 완성된 이메일 제공!

■ 예시 2: 여행 계획

**나쁜 프롬프트:**
```
여행 계획 짜줘
```

**좋은 프롬프트:**
```
제주도 2박 3일 여행 계획을 짜줘.
- 여행 시기: 12월 말
- 인원: 성인 2명
- 예산: 50만원 (숙소, 식사 포함)
- 선호: 자연 경관, 맛집 탐방
- 이동 수단: 렌터카
```

---

## 좋은 프롬프트의 5가지 원칙

**1. 구체적으로 말해**
- ❌ "글 써줘"
- ✅ "SNS에 올릴 신제품 홍보 문구 3개 써줘"

**2. 맥락을 알려줘**
- ❌ "요약해줘"
- ✅ "이 기사를 중학생도 이해할 수 있게 3줄로 요약해줘"

**3. 원하는 형식을 지정해**
- ❌ "비교해줘"
- ✅ "아이폰과 갤럭시를 표 형식으로 비교해줘 (가격, 카메라, 배터리 기준)"

**4. 역할을 부여해**
- ❌ "조언해줘"
- ✅ "너는 10년차 마케터야. 스타트업 브랜딩 전략을 조언해줘"

**5. 예시를 보여줘**
- ❌ "제목 만들어줘"
- ✅ "유튜브 썸네일 제목 만들어줘. 예시: '이것만 알면 끝!' 같은 호기심 유발 스타일로"

## 바로 쓰는 프롬프트 템플릿

복사해서 바로 사용해봐!

■ 글쓰기용 템플릿
```
[주제]에 대한 [형식]을 써줘.
- 타겟 독자: [누구]
- 분량: [몇 글자/몇 문장]
- 톤앤매너: [격식체/친근체/전문적]
- 포함할 내용: [핵심 키워드]
```

■ 요약용 템플릿
```
다음 내용을 [대상]이 이해할 수 있게 [분량]으로 요약해줘.
핵심 포인트 [몇 개]를 불릿으로 정리해줘.

[요약할 내용 붙여넣기]
```

■ 아이디어 브레인스토밍 템플릿
```
[주제/상황]에 대한 아이디어 [개수]개를 제안해줘.
- 조건: [제한사항]
- 스타일: [창의적/실용적/트렌디]
- 예시 참고: [벤치마크 사례]
```

■ 분석용 템플릿
```
[분석 대상]의 장단점을 분석해줘.
- 비교 기준: [항목1, 항목2, 항목3]
- 출력 형식: 표
- 결론에 추천 의견도 포함해줘
```

---

## 실전 활용 팁

> 💡 **꿀팁**: AI 답변이 마음에 안 들면 "다시 해줘"보다 "좀 더 [구체적인 요청]하게 수정해줘"가 훨씬 효과적이야!

**추가 명령어 예시:**
- "좀 더 짧게" / "좀 더 자세하게"
- "좀 더 쉬운 말로"
- "예시를 추가해줘"
- "표로 정리해줘"
- "1번 항목을 더 구체적으로"

---

## 주의사항

- **개인정보 입력 금지**: 주민번호, 비밀번호 등 민감한 정보는 절대 입력하지 마
- **팩트체크 필수**: AI가 틀린 정보를 말할 수 있어. 중요한 건 꼭 확인해봐
- **저작권 주의**: AI가 만든 내용도 표절 검사 해보는 게 좋아
