## ■ Canva AI란?

Canva는 **디자인을 몰라도** 예쁜 이미지를 만들 수 있는 무료 도구입니다.

특히 **Magic Media** 기능을 사용하면 **글만 입력해도 AI가 이미지를 생성**해줍니다!

### 무료 버전 제공 기능
- 월 50회 AI 이미지 생성
- 수천 개 템플릿 무료
- 기본 편집 기능 전부 무료

---

## ■ Canva AI 시작하기

### 1단계: Canva 접속
https://www.canva.com 접속 후 로그인

### 2단계: 디자인 시작
- "디자인 만들기" 클릭
- 원하는 크기 선택
  - 인스타그램 게시물: 1080 x 1080
  - 인스타그램 스토리: 1080 x 1920
  - 배너: 원하는 크기 직접 입력

### 3단계: Magic Media 실행
- 왼쪽 메뉴에서 "앱" 클릭
- "Magic Media" 또는 "AI 이미지" 검색
- 프롬프트 입력 후 "생성" 클릭

## ■ 인스타 피드 이미지 만들기

인스타그램 게시물용 이미지 프롬프트입니다.

### 음식 사진 프롬프트

```
맛있어 보이는 김치찌개 사진, 김이 모락모락,
따뜻한 조명, 한식당 분위기, 음식 사진, 고해상도
```

```
신선한 샐러드 사진, 밝은 자연광,
깔끔한 흰색 접시, 카페 분위기, 음식 사진
```

```
커피 한 잔, 라떼아트, 나무 테이블,
아늑한 카페 분위기, 따뜻한 조명
```

### 공간/인테리어 프롬프트

```
아늑한 카페 인테리어, 따뜻한 조명,
나무 가구, 감성적인 분위기, 인스타그램 스타일
```

```
깔끔한 미용실 인테리어, 밝은 조명,
현대적인 디자인, 전문적인 느낌
```

## ■ 프로모션 배너 만들기

이벤트/할인 홍보용 배너 프롬프트입니다.

### 세일 배너

```
세일 배너, 빨간색과 노란색, "50% 할인" 느낌,
쇼핑 분위기, 밝고 눈에 띄는 디자인
```

### 시즌 배너

```
봄 프로모션 배너, 벚꽃, 파스텔 핑크,
화사하고 밝은 분위기, 깔끔한 디자인
```

```
크리스마스 이벤트 배너, 빨강과 초록,
눈 내리는 분위기, 축제 느낌
```

### 오픈 배너

```
그랜드 오픈 배너, 금색과 검정색,
고급스러운 느낌, 축하 분위기
```

## ■ 업종별 이미지 프롬프트

### 카페

```
아이스 아메리카노, 투명한 유리컵,
얼음이 보이는, 시원한 느낌, 여름 분위기
```

```
따뜻한 카페라떼, 하트 라떼아트,
아늑한 분위기, 겨울 카페
```

### 음식점

```
맛있는 삼겹살 구이, 연기가 피어오르는,
고기집 분위기, 식욕 돋우는 사진
```

```
푸짐한 한정식 상차림, 다양한 반찬,
전통 한식 분위기, 고급스러운 느낌
```

### 미용실

```
세련된 헤어스타일, 여성 모델,
밝은 미용실 배경, 전문적인 느낌
```

```
남성 헤어컷, 깔끔한 스타일,
모던한 바버샵 분위기
```

### 헬스장

```
운동하는 사람 실루엣, 역동적인 포즈,
밝은 조명, 동기부여 느낌
```

```
깔끔한 피트니스 센터, 운동 기구,
현대적인 인테리어, 전문적인 분위기
```

### 꽃집

```
아름다운 장미 꽃다발, 분홍색과 빨간색,
로맨틱한 분위기, 선물 느낌
```

```
화사한 봄 꽃 바구니, 다양한 꽃,
밝은 자연광, 싱그러운 느낌
```

## ■ 이미지 편집 팁

AI로 이미지를 생성한 후 **Canva 편집 기능**으로 다듬으세요.

### 추가할 요소
- 가게 로고
- 이벤트 텍스트 ("50% 할인!")
- 연락처/위치 정보
- 해시태그

### Canva 편집 순서
1. AI 이미지 생성
2. 텍스트 추가 (메뉴 > 텍스트)
3. 로고 업로드 (메뉴 > 업로드)
4. 색상/필터 조정
5. 다운로드 (PNG 또는 JPG)
