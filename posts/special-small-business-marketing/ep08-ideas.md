## ■ 이벤트 아이디어 10개 받기

이벤트 기획, 혼자 고민하지 마세요. AI에게 물어보면 됩니다!

### 프롬프트

```
[업종] 가게에서 할 수 있는 이벤트 아이디어 10개를 알려줘.

가게 정보:
- 업종: [업종]
- 타겟 고객: [예: 20-30대 직장인]
- 예산: [예: 50만원 이내]
- 기간: [예: 1주일]

조건:
- 실현 가능한 아이디어
- 고객 참여 유도
- SNS 바이럴 요소 포함
- 예상 비용도 함께 알려줘
```

### 예시 결과

```
[카페 이벤트 아이디어 10가지]

1. 출석 도장 이벤트
   - 5회 방문 시 음료 1잔 무료
   - 예산: 5만원/월
   - SNS 인증 시 도장 2개

2. 인스타 팔로우 할인
   - 팔로우 인증 시 10% 할인
   - 예산: 거의 없음
   - 팔로워 증가 효과

3. 생일 축하 이벤트
   - 생일자 음료 50% 할인
   - 예산: 10만원/월
   - 생일 인증 필요

4. 비 오는 날 이벤트
   - 우천 시 아메리카노 500원 할인
   - 예산: 날씨에 따라 변동
   - 비 오는 날 매출 증가

5. 리뷰 작성 이벤트
   - 네이버 리뷰 작성 시 쿠키 증정
   - 예산: 10만원/월
   - 리뷰 수 증가 효과

(후략)
```

## ■ 월간 마케팅 캘린더 만들기

한 달 마케팅 계획을 한눈에 정리하세요.

### 프롬프트

```
[업종] 가게의 [월] 마케팅 캘린더를 만들어줘.

포함할 내용:
- 해당 월의 시즌 이벤트 (발렌타인, 어버이날 등)
- 날씨/계절 관련 마케팅
- SNS 게시물 주제 (주 2~3회)
- 추천 프로모션

표 형식으로 정리해줘.
```

### 예시 결과 (2월 카페)

```
[2월 카페 마케팅 캘린더]

| 주차 | 시즌 이벤트 | SNS 주제 | 프로모션 |
|------|-----------|---------|---------|
| 1주 | 설날 연휴 | 새해 인사 | 연휴 영업 안내 |
| 2주 | 발렌타인 준비 | 초콜릿 음료 소개 | 2인 세트 할인 |
| 3주 | 발렌타인데이 | 커플 이벤트 | 하트 라떼아트 |
| 4주 | 봄 준비 | 신메뉴 예고 | 봄 시즌 음료 출시 |

[주간 SNS 게시물 아이디어]
- 월: 주간 인사 + 이번 주 이벤트 안내
- 수: 메뉴 소개 또는 제조 과정
- 금: 주말 이벤트 예고 또는 고객 리뷰 공유
```

## ■ 경쟁 분석 요청하기

우리 가게의 문제점과 해결책을 AI에게 물어보세요.

### 프롬프트

```
[업종]에서 효과적인 마케팅 전략이 뭐가 있는지 알려줘.

우리 가게 상황:
- 위치: [위치]
- 오픈 기간: [기간]
- 현재 문제점: [예: 재방문율이 낮음]

다음 내용을 분석해줘:
1. 이 업종에서 많이 쓰는 마케팅 전략 5가지
2. 우리 가게 문제점의 원인 분석
3. 구체적인 해결 방안 3가지
4. 차별화할 수 있는 포인트
```

### 예시 결과

```
[재방문율 낮은 카페 분석]

## 1. 카페 업종 주요 마케팅 전략
- 멤버십/스탬프 프로그램
- SNS 이벤트 (인증샷 할인)
- 시즌 한정 메뉴
- 굿즈 판매
- 공간 대여 서비스

## 2. 재방문율 낮은 원인 분석
- 경쟁 카페가 많은 지역 특성
- 차별화 포인트 부족
- 재방문 유인 장치 없음
- 고객 데이터 미활용

## 3. 해결 방안
1) 스탬프 카드 도입 (5잔 마시면 1잔 무료)
2) 단골 고객 전용 메뉴/할인 제공
3) 생일/기념일 자동 쿠폰 발송

## 4. 차별화 포인트
- 특정 시간대 할인 (오전 7-9시 직장인 타임)
- 노트북 작업하기 좋은 환경 강조
- 원두 구독 서비스 도입
```

## ■ 시즌별 마케팅 전략

연간 마케팅 포인트를 미리 파악하세요.

### 프롬프트

```
[업종] 가게의 연간 시즌 마케팅 전략을 알려줘.

조건:
- 월별 주요 이벤트/기념일
- 해당 시즌에 맞는 마케팅 아이디어
- 매출이 낮은 시기 대응 전략
- 표 형식으로 정리
```

| 3월 | 졸업/입학 | 단체 예약 할인 | - |
| 4월 | 봄/벚꽃 | 봄 신메뉴 | - |
| 5월 | 어버이날 | 가족 메뉴 | 성수기 |
| 6월 | - | 비수기 프로모션 | 비수기 |
| 7월 | 휴가철 | 시원한 메뉴 | - |
| 8월 | 휴가철 | 보양식 | - |
