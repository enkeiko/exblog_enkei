## ■ 왜 인스타그램인가요?

**인스타그램 마케팅이 중요한 이유:**

1. **검색 플랫폼 역할**: 요즘 MZ세대는 네이버보다 인스타로 맛집 검색
2. **시각적 어필**: 음식, 인테리어 사진으로 고객 유입
3. **무료 광고**: 해시태그로 무료 노출
4. **신뢰도 형성**: 활발한 계정 = 운영 잘 되는 가게

**하지만 현실은...**
- 뭘 써야 할지 모르겠음
- 매번 비슷한 글만 올림
- 해시태그 찾기 귀찮음

AI가 이 모든 걸 해결해드립니다!

```

작성 조건:
- 이모지 적절히 사용

- 친근하고 가벼운 말투
- 150자 내외
```

## ■ 업종별 맞춤 프롬프트

### 음식점용

```
[가게명] 인스타그램 게시글 작성해줘.

정보:
- 오늘의 추천 메뉴: [메뉴명]
- 특징: [예: 8시간 푹 고은 사골육수]
- 가격: [가격]
- 위치: [위치]

조건:
- 음식 관련 이모지 많이 사용
- 맛집 해시태그 포함
- 식욕 돋우는 표현
- 150자 내외
```

### 미용실용

```
[미용실명] 인스타그램 게시글 작성해줘.

정보:
- 이번 달 이벤트: [예: 펌+컷 20% 할인]
- 기간: [기간]
- 예약 방법: [예: DM 또는 전화]

조건:
- 고객들이 예약하고 싶게 만드는 문구
- 뷰티 관련 이모지 사용
- 해시태그 10개
- 150자 내외
```

### 헬스장/필라테스용

```
[센터명] 인스타그램 게시글 작성해줘.

정보:
- 프로그램: [예: 1:1 PT 체험권]
- 특징: [예: 체형 분석 무료 제공]
- 가격: [가격]

조건:
- 운동 의욕 불타오르게 하는 문구
- 피트니스 이모지 사용
- 동기부여 느낌
- 해시태그 10개
```

### 꽃집용

```
[꽃집명] 인스타그램 게시글 작성해줘.

정보:
- 오늘의 꽃: [예: 장미, 튤립]
- 이벤트: [예: 꽃다발 10% 할인]
- 배달 가능 여부: [가능/불가능]

조건:
- 감성적이고 따뜻한 문구
- 꽃/식물 이모지 사용
- 선물 추천 느낌
- 해시태그 10개
```

## ■ 해시태그 자동 생성 팁

해시태그만 따로 받고 싶을 때 사용하세요.

```
[업종] [지역] 관련 인스타그램 해시태그 20개 만들어줘.

조건:
- 대형 해시태그 5개 (게시물 100만 이상)
- 중형 해시태그 10개 (게시물 1만~100만)
- 소형 해시태그 5개 (게시물 1만 이하)
```

**예시 결과:**
```
대형: #맛집 #카페 #서울맛집 #카페스타그램 #먹스타그램
중형: #강남카페 #강남맛집 #신메뉴 #카페추천 #커피스타그램 ...
소형: #강남역카페추천 #강남신상카페 #강남데이트코스 ...
```
