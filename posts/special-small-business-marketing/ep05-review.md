## ■ 리뷰 답변, 왜 중요할까요?

**리뷰 답변의 효과:**

1. **재방문율 증가**: 답변 받은 고객은 다시 올 확률 높음
2. **신규 고객 유입**: 다른 고객들이 답변 보고 신뢰도 판단
3. **부정 리뷰 완화**: 진심 어린 답변은 별점 수정으로 이어지기도
4. **검색 노출**: 답변도 검색에 반영됨

**하지만 현실은...**
- 하루에 리뷰가 5개, 10개씩 쌓임
- 뭐라고 써야 할지 모르겠음
- 부정 리뷰는 더 어려움

AI가 해결해드립니다!

## ■ 긍정 리뷰 답변 프롬프트

칭찬 리뷰에도 정성스러운 답변이 필요합니다.

```
다음 고객 리뷰에 대한 답변을 작성해줘.

업종: [업종]
리뷰: "[고객 리뷰 내용]"

조건:
- 친근하고 따뜻한 경어체
- 리뷰 내용을 구체적으로 언급
- 재방문 유도 문구
- 2~3문장
- 이모지 1~2개
```

### 예시

**입력:**
```
업종: 파스타집
리뷰: "파스타 정말 맛있었어요! 분위기도 좋고 직원분도 친절하셨습니다."
```

**AI 출력:**
```
안녕하세요! 저희 파스타를 맛있게 드셨다니 정말 기뻐요 😊
분위기와 서비스까지 만족하셨다니 힘이 납니다.
다음에 오시면 새로 나온 리조또도 추천드릴게요. 또 뵙겠습니다! 🍝
```

## ■ 부정 리뷰 답변 프롬프트 (중요!)

부정 리뷰 대응이 가장 중요합니다. 잘못 대응하면 더 큰 문제가 됩니다.

```
다음 부정적인 고객 리뷰에 대한 답변을 작성해줘.

업종: [업종]
리뷰: "[불만 내용]"

조건:
- 진심으로 사과하는 톤
- 절대 변명하지 않기
- 개선 의지 표현
- 연락처 또는 재방문 시 보상 언급
- 3~4문장
- 경어체
```

### 예시

**입력:**
```
업종: 삼겹살집
리뷰: "음식이 너무 짜고, 서빙도 느렸어요. 실망이에요."
```

**AI 출력:**
```
안녕하세요, 먼저 불편을 드려 정말 죄송합니다 🙏
음식 간과 서빙 속도 모두 바로 점검하겠습니다.
다음에 방문해주시면 더 나은 모습으로 보답드릴게요.
혹시 추가로 말씀해주실 내용 있으시면 010-XXXX-XXXX로 연락주세요.
```

### 부정 리뷰 답변 핵심 원칙

| 해야 할 것 | 하면 안 되는 것 |
|-----------|----------------|
| 진심으로 사과 | 변명하기 |
| 개선 약속 | 고객 탓하기 |
| 연락처 제공 | 무시하기 |
| 재방문 유도 | 감정적 대응 |

## ■ 리뷰 10개 한번에 처리하기

리뷰가 많이 쌓였다면? 한번에 처리하세요!

```
다음 리뷰들에 대한 답변을 각각 작성해줘.

업종: [업종]

리뷰 목록:
1. "[리뷰 1]"
2. "[리뷰 2]"
3. "[리뷰 3]"
4. "[리뷰 4]"
5. "[리뷰 5]"

조건:
- 각 답변은 2~3문장
- 친근한 경어체
- 리뷰 내용 구체적으로 언급
- 이모지 1개씩 포함
```

### 예시

**입력:**
```
업종: 삼겹살집

리뷰 목록:
1. "고기 굽는 냄새 안 배서 좋았어요"
2. "반찬이 좀 아쉬웠어요"
3. "가성비 최고! 또 올게요"
4. "웨이팅이 길었지만 맛있어서 참았어요"
5. "직원분이 고기 구워주셔서 편했어요"

각 답변 2~3문장, 경어체로 작성해줘.
```

**AI 출력:**
```
1. 안녕하세요! 환기 시스템에 신경 많이 썼는데 만족하셨다니 기뻐요 😊 또 편하게 방문해주세요!

2. 안녕하세요, 반찬이 아쉬우셨군요. 죄송합니다 🙏 더 다양하고 맛있는 반찬 준비하겠습니다. 다음에 오시면 달라진 모습 보여드릴게요!

3. 가성비 좋게 봐주셔서 감사합니다! 💪 다음에도 푸짐하게 준비해서 기다릴게요. 꼭 또 와주세요!

4. 웨이팅이 길었는데도 기다려주셔서 정말 감사해요 🙏 더 빠른 서비스로 보답하겠습니다!

5. 직접 구워드린 게 도움이 됐다니 다행이에요 😊 다음에도 맛있게 구워드릴게요!
```
