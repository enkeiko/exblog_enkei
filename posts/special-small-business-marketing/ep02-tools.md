## ■ ChatGPT 가입 방법

ChatGPT는 가장 유명한 AI 도구입니다. 글쓰기, 아이디어 브레인스토밍, 번역 등 거의 모든 작업이 가능해요.

### 가입 순서

**1단계**: https://chat.openai.com 접속

**2단계**: "Sign up" 버튼 클릭

**3단계**: 가입 방법 선택
- 구글 계정으로 가입 (추천 - 가장 빠름)
- 이메일로 가입
- 애플 계정으로 가입

**4단계**: 가입 완료!

### ChatGPT 특징

**장점**
- 가장 범용적인 AI
- 한국어 이해력 우수
- 무료 버전도 충분히 유용

**무료 버전 제한**
- GPT-4o mini 모델 사용
- 하루 메시지 수 제한 있음
- 이미지 생성 불가

## ■ Claude 가입 방법

Claude는 Anthropic에서 만든 AI입니다. **긴 글 작성**이나 **문서 분석**에 특히 강해요.

### 가입 순서

**1단계**: https://claude.ai 접속

**2단계**: "Start for free" 또는 "Get started" 클릭

**3단계**: 가입 방법 선택
- 구글 계정으로 가입 (추천)
- 이메일로 가입

**4단계**: 가입 완료!

### Claude 특징

**장점**
- 긴 문서 처리에 강함
- 논리적이고 체계적인 답변
- 한국어 품질 우수

**무료 버전 제한**
- 하루 대화 횟수 제한 (약 30~50회)
- 파일 업로드 크기 제한

---

## ■ Canva 가입 방법

Canva는 디자인 도구입니다. 디자인을 몰라도 **AI가 이미지를 만들어줍니다**.

### 가입 순서

**1단계**: https://www.canva.com 접속

**2단계**: "무료로 시작하기" 클릭

**3단계**: 가입 방법 선택
- 구글 계정으로 가입 (추천)
- 이메일로 가입
- 페이스북으로 가입

**4단계**: 가입 완료!

### Canva 특징

**장점**
- 템플릿 수만 개 제공
- AI 이미지 생성 기능 (Magic Media)
- 드래그 앤 드롭으로 쉬운 편집

**무료 버전 제한**
- 월 50회 AI 이미지 생성
- 일부 템플릿 유료

## ■ 내 가게 정보 정리하기

AI에게 좋은 결과물을 받으려면 **내 가게 정보를 미리 정리**해두세요. 이걸 메모장에 저장해두면 매번 입력할 필요가 없습니다.

### 정리 템플릿

```
[내 가게 기본 정보]

• 업종:
• 가게명:
• 위치:
• 영업시간:
• 전화번호:
• 대표 메뉴/상품:
• 가격대:
• 우리 가게만의 특징:
• 타겟 고객:
• SNS 계정:
```

### 예시

```
[내 가게 기본 정보]

• 업종: 카페
• 가게명: 커피하우스
• 위치: 서울 강남역 5번출구 도보 1분
• 영업시간: 평일 8-22시, 주말 10-22시
• 전화번호: 02-1234-5678
• 대표 메뉴: 아메리카노, 카페라떼, 딸기라떼
• 가격대: 4,000~6,000원
• 우리 가게만의 특징: 직접 로스팅한 원두, 넓은 좌석
• 타겟 고객: 20-30대 직장인
• SNS 계정: @coffeehouse_gangnam
```

이 정보를 AI에게 함께 전달하면 **우리 가게에 딱 맞는 맞춤형 콘텐츠**를 만들어줍니다.
