## ■ 프롬프트 저장해서 재사용하기

매번 처음부터 입력하지 마세요. **자주 쓰는 프롬프트는 저장**해두세요!

### 저장 추천 프롬프트

```
[내 가게 인스타 게시글용]

너는 [업종] 가게의 SNS 마케팅 담당자야.
인스타그램 게시글 작성해줘.

가게: [가게명], [위치]
홍보 내용: (여기만 바꾸기)

조건: 이모지 사용, 해시태그 10개, 친근한 경어체, 150자
```

```
[내 가게 리뷰 답변용]

다음 고객 리뷰에 대한 답변 작성해줘.

업종: [업종]
리뷰: (여기에 붙여넣기)

조건: 친근한 경어체, 리뷰 내용 언급, 재방문 유도, 2~3문장
```

### 저장 위치 추천
- **메모장**: 가장 간단
- **노션**: 카테고리별 정리 가능
- **구글 문서**: 어디서든 접근 가능
- **카카오톡 나에게 보내기**: 스마트폰에서 바로 사용

## ■ AI 결과물 다듬는 팁

AI가 만든 글은 **70~80% 완성본**입니다. 마지막 20%는 사장님이 다듬어야 해요.

### 꼭 확인할 것

| 항목 | 확인 내용 |
|------|----------|
| **정보 정확성** | 가격, 위치, 시간 등 틀린 거 없는지 |
| **가게 개성** | 우리 가게만의 표현이 들어갔는지 |
| **어색한 표현** | 너무 AI스러운 문장은 없는지 |
| **해시태그** | 우리 지역/업종에 맞는지 |

### 수정 예시

**AI 원본:**
```
저희 가게는 최고의 맛을 제공합니다.
```

**수정 후:**
```
10년째 같은 자리에서 단골분들과 함께합니다.
```

**포인트**: "최고", "최상" 같은 추상적 표현 → 구체적인 스토리로 변경

## ■ 일주일 콘텐츠 한번에 만들기

매일 고민하면 지칩니다. **일주일치를 한번에** 만들어두세요!

### 일괄 생성 프롬프트

```
이번 주 인스타그램 게시글 7개 만들어줘.

가게: [가게명], [업종], [위치]

요일별 주제:
- 월: 주간 인사 + 이번 주 이벤트
- 화: 대표 메뉴 소개
- 수: 고객 리뷰 감사 인사
- 목: 비하인드 스토리 (준비 과정 등)
- 금: 주말 이벤트 예고
- 토: 주말 영업 안내
- 일: 다음 주 예고

각각 150자 이내, 해시태그 5개씩 포함, 경어체로 작성해줘.
```

### 예약 발행 활용

인스타그램/페이스북은 **예약 발행** 기능이 있습니다.
일주일치 만들어서 미리 예약해두세요!

## ■ 주의사항 4가지 (중요!)

### 1. AI 글 100% 그대로 쓰지 마세요

**문제점:**
- 다른 가게도 AI 쓰면 비슷한 글이 나옴
- 진정성이 느껴지지 않음
- 고객이 AI 글이라는 걸 알아챔

**해결책:**
- 우리 가게만의 표현 추가
- 실제 경험/에피소드 넣기
- 지역 특성 반영

### 2. 허위/과장 광고 주의

**위험한 표현:**
- "최고의 맛" (증명 불가)
- "100% 만족 보장" (불가능한 약속)
- "업계 1위" (근거 없으면 위법)

**안전한 표현:**
- "정성껏 준비한"
- "많은 분들이 좋아하시는"
- "10년 전통의"

### 3. 저작권 이미지 주의

| 사용 가능 | 주의 필요 |
|----------|----------|
| Canva AI 생성 이미지 | 인터넷에서 가져온 이미지 |
| 직접 촬영한 사진 | 다른 블로그 이미지 |
| Unsplash 무료 이미지 | 출처 불명확한 이미지 |

**추천:** 직접 찍은 사진 + AI 편집 조합

### 4. 개인정보 입력 금지

AI에 **절대 입력하면 안 되는 것:**
- 고객 전화번호, 주소
- 가게 매출, 원가 정보
- 직원 개인정보
- 계약서 내용
