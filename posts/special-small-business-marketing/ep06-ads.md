## ■ 카카오톡 채널 메시지

카카오톡 채널 메시지는 **짧고 임팩트 있게** 써야 합니다.

### 프롬프트

```
[가게명] 카카오톡 채널 메시지를 작성해줘.

목적: [예: 오픈 1주년 이벤트 홍보]
이벤트 내용: [예: 전 메뉴 20% 할인]
기간: [기간]

조건:
- 100자 이내
- 행동 유도 문구 포함 (예: 지금 바로 예약하세요!)
- 이모지 2~3개
- 경어체
```

### 예시 결과

```
🎉 커피하우스 1주년 감사 이벤트!

전 메뉴 20% 할인 (12/1~12/7)
선착순 100명 한정!

📍 강남역 5번출구 1분
지금 바로 방문하세요! ☕
```

## ■ 배민/요기요 가게 소개글

배달앱 소개글은 **첫 주문을 유도**하는 게 핵심입니다.

### 프롬프트

```
배달앱에 올릴 가게 소개글을 작성해줘.

가게명: [가게명]
대표 메뉴: [메뉴 3가지]
우리 가게만의 특징: [예: 매일 아침 손수 만드는 수제 소스]
배달 가능 지역: [지역]
첫 주문 혜택: [예: 음료 서비스]

조건:
- 300자 내외
- 음식 관련 이모지 사용
- 첫 주문 고객 혜택 강조
- 경어체
```

### 예시 결과

```
🍝 정통 이탈리안 파스타 전문점

안녕하세요! 매일 아침 직접 만드는 수제 토마토소스로
정성껏 요리합니다.

✨ 대표 메뉴
• 크림 파스타 - 진한 크림의 부드러움
• 토마토 파스타 - 상큼한 토마토의 풍미
• 로제 파스타 - 크림과 토마토의 환상 조합

🎁 첫 주문 고객님께 음료 서비스!

📍 강남구 전 지역 배달 가능
⏰ 평일 11:00~22:00

맛있는 파스타로 행복한 식사 되세요! 🍴
```

## ■ 현수막/배너 문구

현수막은 **한눈에 들어오는 짧은 문구**가 핵심입니다.

### 프롬프트

```
가게 앞에 걸 현수막 문구 5개 만들어줘.

업종: [업종]
홍보 내용: [예: 점심 특선 6,900원]
타겟: [예: 직장인]

조건:
- 각 10~15글자 이내
- 한눈에 들어오게
- 가격 강조
- 행동 유도
```

### 예시 결과

```
1. 점심특선 6,900원! 든든하게 드세요
2. 런치 특가! 매일 6,900원
3. 직장인 점심 해결사 6,900원
4. 이 가격 실화? 점심특선 6,900원
5. 배부르게! 맛있게! 6,900원
```

### 업종별 현수막 예시

**카페**
```
- 아아 2,500원! 출근길 한 잔
- 커피 한 잔의 여유 ☕
- 직접 로스팅한 신선한 원두
```

**미용실**
```
- 커트+펌 20% 할인 중!
- 변신의 시작, 여기서부터
- 스타일 업그레이드 찬스!
```

**헬스장**
```
- PT 1회 무료 체험!
- 올해는 꼭! 운동 시작
- 체형분석 무료 상담
```

## ■ 전단지 문구

전단지는 **정보 + 혜택 + 행동유도**가 필요합니다.

### 프롬프트

```
[가게명] 전단지 문구를 작성해줘.

가게 정보:
- 업종: [업종]
- 위치: [위치]
- 전화: [전화번호]
- 영업시간: [시간]

홍보 내용: [이벤트/메뉴]
타겟: [타겟 고객]

조건:
- 헤드라인 (10글자 이내)
- 서브 카피 (20글자 이내)
- 본문 (3~5줄)
- 행동 유도 문구
```

### 예시 결과

```
[헤드라인]
오픈 기념 50% 할인!

[서브 카피]
강남역 신상 삼겹살 맛집 탄생

[본문]
✅ 국내산 생삼겹살
✅ 무한리필 반찬
✅ 넓은 단체석 완비
✅ 주차 가능

[혜택]
이 전단지 지참 시 소주 1병 서비스!

[정보]
📍 강남역 5번출구 3분
📞 02-1234-5678
⏰ 11:00~24:00
```

## ■ 광고 문구 작성 체크리스트

좋은 광고 문구의 조건:

- [ ] **한눈에 들어오는가?** (10초 안에 이해)
- [ ] **혜택이 명확한가?** (할인, 서비스 등)
- [ ] **행동 유도가 있는가?** (방문하세요, 주문하세요)
- [ ] **타겟에 맞는가?** (직장인, 학생, 가족 등)
- [ ] **차별점이 있는가?** (우리만의 특징)
