## ■ 왜 네이버 블로그인가요?

**한국에서 네이버 블로그가 중요한 이유:**

1. **검색 점유율 1위**: 한국인 70% 이상이 네이버 검색 사용
2. **지역 검색**: "강남역 맛집" 검색 시 블로그가 상위 노출
3. **신뢰도**: 블로그 후기 = 실제 방문 후기로 인식
4. **무료 광고**: 잘 쓴 글 하나가 수년간 고객 유입

**문제는...**
- 글쓰기가 어려움
- SEO가 뭔지 모름
- 시간이 너무 오래 걸림

AI가 해결해드립니다!

## ■ SEO 최적화 블로그 프롬프트

이 프롬프트 하나면 네이버 상위노출을 노릴 수 있습니다.

```
[가게명] 네이버 블로그 포스팅을 작성해줘.

가게 정보:
- 업종: [업종]
- 가게명: [가게명]
- 위치: [상세 위치]
- 영업시간: [시간]
- 주차: [가능/불가능]
- 대표 메뉴: [메뉴 3가지]

소개할 내용: [예: 시그니처 메뉴 3가지 소개]

SEO 조건:
- 제목에 "[지역명] + [업종]" 키워드 포함
- 첫 문장에 핵심 키워드 포함
- 소제목 4~5개 (키워드 포함)
- 본문 1,500자 이상
- 친근한 경어체
- 마지막에 방문 유도 문구
- 해시태그 10개
```

### 실제 결과 예시

**입력:**
```
커피하우스 네이버 블로그 포스팅 작성해줘.

업종: 카페
위치: 서울 강남역 5번출구 도보 1분
영업시간: 평일 8-22시
주차: 불가능 (인근 공영주차장 이용)
대표 메뉴: 아메리카노, 카페라떼, 딸기라떼

소개할 내용: 직접 로스팅하는 원두 이야기

SEO 조건 적용해서 1,500자 이상으로 작성해줘.
```

**AI 출력 (일부):**
```
# 강남역 카페 추천 - 직접 로스팅하는 커피하우스

강남역 카페 찾고 계신가요? 오늘은 강남역 5번출구에서
도보 1분 거리에 있는 '커피하우스'를 소개해드릴게요.

## ■ 커피하우스만의 특별함

커피하우스는 매일 아침 직접 원두를 로스팅합니다...
(후략)
```

## ■ 고객 리뷰 활용 글쓰기

실제 고객 리뷰를 활용하면 더 진정성 있는 글이 됩니다!

```
다음 고객 리뷰들을 바탕으로 블로그 글을 작성해줘.

가게: [가게명], [위치]

고객 리뷰:
1. "[리뷰 내용 1]"
2. "[리뷰 내용 2]"
3. "[리뷰 내용 3]"

조건:
- 리뷰들의 공통점을 자연스럽게 녹여서
- 처음 방문하는 사람에게 우리 가게 장점 소개
- 1,500자 이상
- 경어체
```

### 예시

```
다음 고객 리뷰들을 바탕으로 블로그 글을 작성해줘.

가게: 김사장 김치찌개, 신림역

고객 리뷰:
1. "여기 김치찌개 진짜 맛있어요 국물이 시원해요"
2. "직원분들 친절하셔서 좋았어요"
3. "주차도 편하고 아이들 데려가기 좋아요"

이 리뷰들의 공통점을 살려서 작성해줘.
```

## ■ 키워드 선정 팁

네이버 상위노출의 핵심은 **키워드**입니다.

### 키워드 찾기 프롬프트

```
[업종] [지역] 관련 네이버 블로그 키워드를 추천해줘.

조건:
- 메인 키워드 3개 (검색량 많은 것)
- 세부 키워드 5개 (경쟁 낮은 것)
- 롱테일 키워드 5개 (구체적인 것)
```

### 예시 결과

```
메인 키워드:
- 강남역 카페
- 강남 카페 추천
- 강남역 맛집

세부 키워드:
- 강남역 조용한 카페
- 강남 디저트 카페
- 강남역 노트북 카페
- 강남 데이트 카페
- 강남역 브런치 카페

롱테일 키워드:
- 강남역 5번출구 카페 추천
- 강남역 근처 조용한 카페
- 강남 직장인 점심 카페
- 강남역 미팅하기 좋은 카페
- 강남 루프탑 카페
```
