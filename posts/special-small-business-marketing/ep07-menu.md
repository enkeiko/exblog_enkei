## ■ 메뉴판 설명 작성

좋은 메뉴 설명은 **읽기만 해도 침이 고이게** 만듭니다.

### 프롬프트

```
다음 메뉴의 설명을 작성해줘.

메뉴명: [예: 한우 육회비빔밥]
주재료: [예: 한우 육회, 참기름, 고추장]
특징: [예: 매일 아침 냉장 배송받는 신선한 한우]
가격: [가격]

조건:
- 30~50자
- 식욕 돋우는 표현 사용
- 고급스러운 느낌
- 경어체
```

### 예시 결과

```
메뉴: 한우 육회비빔밥

매일 아침 도착하는 신선한 한우 육회에
고소한 참기름과 비법 고추장을 더해
한 그릇에 담아낸 프리미엄 비빔밥입니다.
```

### 업종별 메뉴 설명 예시

**한식**
```
된장찌개: 3일간 숙성한 집된장으로 깊은 맛을 낸
         구수하고 진한 전통 된장찌개

김치찌개: 6개월 묵은 김치와 국내산 돼지고기가 만나
         얼큰하고 칼칼한 깊은 맛

갈비탕: 10시간 정성껏 고아낸 진한 육수에
       부드러운 갈비살이 가득
```

**카페**
```
아메리카노: 매일 로스팅한 원두로 추출한
           깔끔하고 깊은 풍미의 에스프레소

카페라떼: 부드러운 우유 거품 위에
         진한 에스프레소가 어우러진 한 잔

딸기라떼: 신선한 딸기와 부드러운 우유가 만나
         달콤상큼한 핑크빛 음료
```

## ■ 배달앱 메뉴 설명

배달앱은 **사진 없이 글만 보고 주문**하는 경우도 많습니다.

### 프롬프트

```
배달앱에 올릴 메뉴 설명을 작성해줘.

메뉴명: [메뉴명]
구성: [예: 1인분, 밥 포함]
재료: [주재료]
맵기: [안매움/보통/매움]
추천 대상: [예: 매운 거 좋아하시는 분]

조건:
- 50자 내외
- 구성 명확하게
- 맛 표현 포함
- 이모지 1~2개
```

### 예시 결과

```
🔥 불닭볶음면 (1인분)
매콤한 양념에 쫄깃한 면발!
반숙계란 + 모짜렐라 치즈 토핑
매운맛 마니아를 위한 선택 🌶️
```

## ■ 스마트스토어 상품 설명

온라인 판매는 **상세페이지가 곧 영업사원**입니다.

### 프롬프트

```
네이버 스마트스토어에 올릴 상품 설명을 작성해줘.

상품명: [상품명]
카테고리: [카테고리]
특징: [특징 3가지]
가격: [가격]
배송: [배송 정보]

조건:
- 1,000자 이상
- 소제목으로 구분
- 구매를 부르는 문구
- SEO 키워드 자연스럽게 포함
- 경어체
```

### 상품 설명 구조

```
1. 헤드라인 (한 줄로 핵심 어필)
2. 이런 분께 추천해요 (타겟 명시)
3. 상품 특징 (3가지)
4. 상세 정보 (스펙, 구성)
5. 사용 방법 / 보관 방법
6. 배송 안내
7. 교환/반품 안내
8. 구매 유도 문구
```

### 예시 결과 (일부)

```
# 집에서 즐기는 프리미엄 한우 육회

## 이런 분께 추천드려요
✅ 신선한 육회를 집에서 드시고 싶은 분
✅ 특별한 날 근사한 한 끼를 준비하시는 분
✅ 배달 육회가 불안하셨던 분

## 상품 특징

### 1. 당일 도축, 당일 배송
주문 당일 도축한 신선한 한우만 사용합니다.
냉장 상태로 산지에서 바로 출발!

### 2. 1++ 최상급 한우
한우 등급 중 최상위 1++등급만 엄선했습니다.
부드럽고 고소한 맛이 다릅니다.

### 3. 간편한 구성
육회 + 참기름 + 양념장 + 계란노른자
집에서 간편하게 드실 수 있어요.

(후략)
```

## ■ 가격 표현의 심리학

같은 가격도 **표현 방법에 따라** 느낌이 달라집니다.

### 가격 표현 프롬프트

```
다음 가격을 매력적으로 표현해줘.

상품: [상품명]
원래 가격: [원가]
할인 가격: [할인가]
할인율: [%]

조건:
- 3가지 버전으로 작성
- 심리적으로 저렴해 보이게
- 구매 유도 문구 포함
```

### 가격 표현 팁

| 기존 표현 | 개선 표현 |
|----------|----------|
| 10,000원 | 9,900원 (100원 차이지만 심리적으로 저렴) |
| 30% 할인 | 10,000원 → 7,000원 (3,000원 절약!) |
| 배송비 3,000원 | 무료배송 (3만원 이상) |
| 1개 5,000원 | 2개 9,000원 (1개당 4,500원) |

## ■ 메뉴 설명 체크리스트

좋은 메뉴 설명의 조건:

- [ ] **재료가 명확한가?** (무엇으로 만들었는지)
- [ ] **맛이 상상되는가?** (고소한, 매콤한, 달콤한)
- [ ] **양이 예상되는가?** (1인분, 2~3인분)
- [ ] **특별함이 있는가?** (직접, 수제, 엄선)
- [ ] **읽기만 해도 먹고 싶은가?**
